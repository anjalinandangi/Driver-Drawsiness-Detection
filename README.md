# 🚗 Driver Drowsiness Detection System

A real-time Driver Drowsiness Detection System that uses Computer Vision and Machine Learning to monitor a driver's alertness through facial features and trigger alerts when signs of drowsiness are detected.

## 🚀 Project Overview

This project helps reduce road accidents by continuously monitoring the driver's eye movements and facial landmarks using a webcam. When prolonged eye closure is detected, the system raises an alert to wake the driver.

## ✨ Features

- 👁️ Real-time eye detection
- 😴 Drowsiness detection using facial landmarks
- 🔔 Instant alarm when drowsiness is detected
- 📹 Live webcam monitoring
- 📊 Historical data tracking
- 💻 Simple and user-friendly interface

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/anjalinandangi/Driver-Drowsiness-Detection.git
```

Navigate to the project folder:

```bash
cd Driver-Drowsiness-Detection
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app_main.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
